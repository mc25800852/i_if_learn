# Iterative Influential Features Unsupervised Learning for Complex Data

This package implements the **Iterative Influential Features (i-if-learn)** algorithm proposed in our paper.  
It is designed for **unsupervised learning and clustering in complex, high-dimensional datasets**,  
with an emphasis on identifying influential features during the learning process.

## Installation

To install the package, you can use `pip`. To install it in editable mode for development, run:

```bash
pip install .
```

Alternatively, you can install it from GitHub using the following command:

```bash
pip install git+https://github.com/mc25800852/i_if_learn.git
```

## Method: `i_if_learn()`

The core method of this package is `i_if_learn(X, K, labels, prev_influential_indices, ksp, random_results, component, ...)`. Below are the key inputs and expected data formats:

### **Input Requirements:**

* **`X`**: A numeric data matrix of shape *(n\_samples, n\_features)*. Each row corresponds to a sample, and each column to a feature.

* **`K`**: The number of target clusters or classes.

* **`labels`**, **`prev_influential_indices`**, **`ksp`**: These are outputs from the `if_pca()` preprocessing step:

  * `labels`: Cluster labels predicted by `if_pca()`.
  * `prev_influential_indices`: Initial influential features selected by `if_pca()`.
  * `ksp`: Kruskal-Wallis p-values for each feature (used to assess feature relevance).

* **`random_results`**: A list of F-statistics generated by `multiple_random_sampling()`, used for calibrating statistical thresholds.

* **`threshold_method`** *(default: `'hct'`)*: Specifies how to select influential features in each iteration. Default is Higher Criticism Thresholding (HCT).

* **`method`** *(default: `'laplacian'`)*: Dimensionality reduction method. Options:

  * `'laplacian'`: Laplacian Eigenmap (recommended),
  * `'pca'`: Principal Component Analysis.

* **`component`**: Number of dimensions to reduce to. We recommend setting this to `K + 2`.

* **`max_iter`** *(default: `10`)*: Maximum number of iterations.

* **`convergence_threshold`** *(default: `0.10`)*: The method stops if the relative change in selected influential features between iterations falls below this threshold.

The algorithm uses iterative feature selection and clustering refinement based on `ifpca`-normalized inputs. Influential features are refined over multiple rounds until convergence criteria are met.

---

### (Optional) Non-parametric Version

If the input data is not normally distributed or contains outliers, a non-parametric version of the method is supported:

* Uses **Kruskal-Wallis** statistics instead of ANOVA for feature relevance testing.
* Recommended when the assumption of Gaussian-distributed features is not met.

## Usage
```python
from i_IF_learn import *

clusters, prev_influential_indices, ks_data, ksp = if_pca(X, K)

component=K+2

random_results=multiple_random_sampling(X,K,component, prev_influential_indices)

final_indluential_indices, final_labels = i_if_learn(X,K,labels,prev_influential_indices,ksp,random_results,component)
```
