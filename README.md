# i_IF_learn

`i_IF_learn` is a Python package for high-dimensional clustering methods. It provides tools to perform clustering analysis on complex datasets using advanced algorithms.

## Installation

To install the package, you can use `pip`. To install it in editable mode for development, run:

```bash
pip install .

Alternatively, you can install it from GitHub using the following command:

```bash
pip install git+https://github.com/mc25800852/i_IF_learn.git
